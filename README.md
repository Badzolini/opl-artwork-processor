# OPL Artwork Processor


## Table of Contents

## Installation

1. Install [python3 and pip3](https://python.org).

2. Install the required packages:
    ```sh
    pip3 install -r requirements.txt
    ```


## Running the script

1. Run the script:
    ```sh
    python3 main.py <path_to_opl_artwork> <path_to_output_directory> <suffix>
    ```
